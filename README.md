<p>pLink® is a software dedicated for the analysis of chemically cross-linked proteins or protein complexes using mass spectrometry.</p>
<p>pLink 2 is developed as an upgrade of pLink 1. Compared with pLink 1, pLink 2 provides a graphical user interface, and is ~40 times faster with a newly designed index structure. There are also some improvements in the precision.</p>
